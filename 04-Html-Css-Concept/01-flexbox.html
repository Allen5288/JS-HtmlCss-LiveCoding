CSS Flexbox Complete Study Guide

Introduction to Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items flex (expand) to fill additional space or shrink to fit into smaller spaces.

Flex Item Properties

order

The order property specifies the order of a flexible item relative to the other flex items inside the same container.

3
1
4
2

flex-grow

The flex-grow property specifies how much a flex item will grow relative to the other flex items inside the same container.

grow: 1
grow: 2
grow: 1

flex-shrink

The flex-shrink property specifies how much a flex item will shrink relative to the other flex items inside the same container.

shrink: 0
shrink: 1
shrink: 2

flex-basis

The flex-basis property specifies the initial size of a flex item before growing or shrinking.

basis: 100px
basis: 200px
basis: auto

flex (shorthand)

The flex property is a shorthand for flex-grow, flex-shrink, and flex-basis.

flex: 1 1 100px
flex: 2 1 100px
flex: 0 0 auto

align-self

The align-self property specifies the alignment for a specific flex item inside the container, overriding the align-items value.
